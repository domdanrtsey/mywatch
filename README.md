### mysql  自动化巡检脚本

#### 脚本使用前注意事项

> 需要使用root用户执行
> 下载脚本：

1. 使用说明

   ```shell
   1)、多实例下运行此脚本时，要注意区分不同实例的root用户密码与对应端口号
   myuser="root"
   mypasswd="XXXXX"
   myip="localhost"
   myport="3306"
   ```
   
2. 执行完巡检之后，将在脚本所在的路径下生成html巡检结果报告，如下

   192.168.35.244os_mysql_simple_summary.html

3. 巡检项信息如下（其他统计项可根据实际需要自行添加）

   **0)、巡检ip信息**

   **1)、数据库基本信息与binlog参数信息**

   **2)、数据库运行状况信息**

   **3)、数据库一些重要参数配置信息**
